CS Viva Revision

C Programming

What is a pointer?
A variable that stores the address of another variable.
Difference between structure and union?
Structure allocates separate memory for each member, union shares the same memory.
What is recursion?
A function calling itself to solve smaller parts of a problem.
Types of storage classes?
auto, static, register, extern.
Use of malloc() and free()?
malloc() allocates memory dynamically; free() deallocates it.
Function vs Macro?
Functions are compiled and executed; macros are pre-processed and replaced in code.
What is a file in C?
A method to store data permanently using file operations like fopen() and fclose().

DBMS

What is a database?
An organized collection of related data.
What is normalization?
The process to reduce redundancy and improve data integrity.
Primary key vs Foreign key?
Primary key uniquely identifies a record; foreign key links records between tables.
What is SQL?
A language to interact with and manipulate databases (e.g., SELECT, INSERT).
DBMS vs RDBMS?
RDBMS organizes data into tables with relationships, DBMS might not follow that structure.

Networking

Difference between LAN, MAN, and WAN?
LAN covers a local area, MAN a city, and WAN covers large geographic areas.
What is an IP address?
A unique identifier for a device on a network.
What is a router?
A device that forwards data between networks.
What is topology?
The structure or layout of a network (e.g., star, bus, ring).
What is the function of a switch?
It connects devices in a network and forwards data intelligently.

Web Technology

HTML vs CSS?
HTML structures content; CSS styles it.
What are HTML tags?
They are instructions to the browser; examples: <p>, <div>.
What is JavaScript used for?
To add interactivity to web pages.
What are events in JS?
User actions like clicks, hovers, and key presses that trigger code execution.
Types of CSS inclusion?
Inline, internal, and external CSS.

Object-Oriented Programming

What is a class and what is an object?
A class is a blueprint; an object is an instance of that class.
What is inheritance?
It is a mechanism where one class acquires the properties of another.
What is polymorphism?
The ability of one function or method to behave differently based on input.
What are main features of OOP?
Encapsulation, Inheritance, Polymorphism, and Abstraction.
Procedural vs OOP?
Procedural is function-based; OOP is object-based, organizing code around objects.

Bonus Questions

What is SDLC?
Software Development Life Cycle – steps to build software.
What is Cloud Computing?
Storing and accessing data over the internet instead of local systems.
What is Big Data?
Extremely large and complex data sets that require advanced methods to analyze.
What is AI?
Artificial Intelligence – machines simulating human intelligence.
What is IoT?
Internet of Things – network of interconnected devices.
What is Cyber Security?
Protection of systems and data from digital attacks.
